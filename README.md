# nummerino

Convert numbers to German words.

## Usage

### Install

```bash
npm install -g @carlotrimarchi/nummerino
```

### Run
```bash
nummerino <number> [--splitted]
```

## Example

```bash
nummerino 123
# einhundertdreiundzwanzig

nummerino 1234567
# eine million zweihundertvierunddreiĂźigtausendfĂĽnfhundertsiebenundsechzig

nummerino 123 --splitted
# ein hundert drei und zwanzig


```
